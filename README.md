## About the book

This repository contains the companion source code for the book *Setting up a secure logging and metric platform*. The book walks you step by step through deploying, and securing a modern logging and metrics platform. Covering everything from collecting and shipping data, to storing and visualising it, to locking the whole thing down with TLS and authentication.

If you've ever wanted a single, opinionated end-to-end reference for running observability in production, this is for you.

## What's in this repository

This contains all the code that is used in the book. The information in the book describes which file is part of an action.

## Getting started

1. Grab a copy of the book from [Leanpub](https://leanpub.com/settingupasecureloggingandmetrics).
2. Clone this repository:
   ```bash
   git clone git@github.com:dj-wasabi/setting-up-secure-logging-and-metrics-platform-guide.git
   ```
3. Follow along with the chapters — each section of the book references the relevant files here.
